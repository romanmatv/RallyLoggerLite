# RallyLoggerLite
Простая программа для подсчета результатов участника MRC.

## Базовая инструкция
В Трек загружается gpx с Waypoints точками, по которым должен проехать участник
В Заезд загружается gpx с записанным треком участника (OsmAnd, Геотрекер и т.д.)

После выставления базовых настроек и загрузки gpx - программа показывает какие точки участник забрал или не забрал. А так уже показывает на графике с какой скоростью он ехал.
