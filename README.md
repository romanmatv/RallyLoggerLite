# RallyLoggerLite
Простая программа для подсчета результатов участника MRC.

## Базовая инструкция
В Трек загружается gpx с Waypoints точками или проект Rally Navigator 2, по которым должен проехать участник
В Заезд загружается gpx с записанным треком участника (OsmAnd, Геотрекер и т.д.)

После выставления базовых настроек и загрузки gpx - программа показывает какие точки участник забрал или не забрал. А так уже показывает на графике с какой скоростью он ехал.

В контекстном меню каждой точки можно добавить дополнительные точки, удалить её или установить локальное ограничение скорости.

## Компиляция
* Установить npm пакеты `npm install`
* Если есть ссылка на тайлы mapbox или ключ доступа к яндекс картам - то в корне создать файл `.env` с содержимым:
```.env 
MAPBOX_URL=https://ссылка_на_mapbox_tiles/{z}/{x}/{y}
YA_TILES_API=АПИ_КЛЮЧ_ЯНДЕКС_КАРТ
```
* * При отсутствии переменной - этого типа тайлов просто не будет в списке
* Собрать бинарник программы `npm run package`
* Собрать инсталятор `npm run make`
